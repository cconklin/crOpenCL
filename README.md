# crOpenCL

OpenCL Bindings fo Crystal

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crOpenCL:
    github: cconklin/crOpenCL
```

Currently only supports Mac OS


## Usage


```crystal
require "crOpenCL"
```

Might require more than that.
